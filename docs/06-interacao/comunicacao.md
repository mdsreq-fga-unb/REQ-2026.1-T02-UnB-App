# 6.2 Comunicação

## Ferramentas

| Ferramenta | Finalidade |
|------------|-----------|
| [Ex.: Slack / Discord] | Comunicação diária entre membros da equipe |
| [Ex.: Zoom / Meet] | Reuniões de revisão e planejamento de sprints |
| [Ex.: GitHub / Jira] | Gerenciamento do backlog e acompanhamento de tarefas |
